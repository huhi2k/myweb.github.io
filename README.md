# myweb.github.io
Đây là trang web mô phỏng 1 trang web bán hàng sử dụng ngôn ngữ PHP, cụ thể là bán account game liên minh huyền thoại (League of Legends) gồm các chức năng:
+ Người dùng đăng nhập, đăng ký hệ thống
+ Người dùng xem thông tin account, mua account, nạp tiền vào tài khoản
+ Người quản trị đăng nhập với tư cách quản trị viên cùng các chức năng: thêm, xóa, sửa tài khoản, kiểm tra, bảo trì tài khoản và các account được bán
